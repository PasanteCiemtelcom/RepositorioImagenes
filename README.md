# RepositorioImagenes
En este código esta creada la página web en la que se almacenan todas las imagenes de las Normativas ISO, funciona como repositorio para luego enviarlas por Microsoft Teams, usando Power Automate.
Las líneas de código son un HTML simple, junto con un poco de CSS, para darle forma de galería a las imágenes y mantener un orden más limpio para el repositorio.
